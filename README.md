# atividade 03 Programação pra internet 2

Uma CRUD completo, permitindo cadastrar, listar, editar e excluir alunos de um banco de dados através de uma API.

## ⚠️ Importante

O projeto Front-end está configurado para consumir uma API no endereço:
`http://127.0.0.1:3000/alunos`. Para que o projeto funcione corretamente, você precisa ter um servidor rodando nesta porta.
